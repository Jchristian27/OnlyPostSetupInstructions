OnlyPost Setup Instructions

Please follow these instructions carefully and in order. This process should take about 20-30 minutes from start to finish.


Saving the OnlyPost Folder onto Your Desktop

  1. Open the email I sent you containing the blue OnlyPost folder.
  2. Right click the blue OnlyPost folder.
  3. Click 'Save Attachment'
  4. Click the drop down menu to the right of 'Where:' and select 'Desktop' (See image below step 5 for reference).
  5. Click the save button.



Using the Terminal

  1. Click the magnifying glass icon located at the top right of your screen (or hit command+space).
  2. Type ‘Terminal’ into the Spotlight Search bar (do not include quotation marks).
  3. Double click the "terminal" option with the black icon to the left displayed in the Spotlight Search bar.
  4. Drag the Terminal icon (the black box at the bottom right of your screen) and drag it next to your safari icon to add it to your dock.
  5. Click the red button in the upper left area of the terminal window (this is how you will close the terminal window from here on out).
  6. To open a new terminal window as requested in some of the instructions below, all you have to do is click the black terminal icon in your dock.
  7. Watch the embedded video tutorial below to see how to complete this section if you get confused with the written instructions above. Click the gear icon when the video starts playing to change the video quality to 720p if it appears blurry.



Saving the Scripts Folder onto Your Desktop

  1. Open a new terminal window (click here if you forgot how to open a new terminal window -- read step 6).
  2. Copy the red text below and paste it into your terminal window and hit enter.

    3. cd Desktop; git clone https://github.com/Jchristian27/Scripts.git

  3. Close the terminal window when the script is done executing (this will only take a few seconds).

Updating Your macOS

  1. Open a new terminal window (click here if you forgot how to open a new terminal window -- read step 6).
  2. Copy and paste the red text below into the terminal and hit enter.

    3. cd Desktop/Scripts; zsh osx_upgrade.sh

  3. If your operating system is up to date enough to use the OnlyPost script you will see a message that says "Everything is up to date!". If your operating system is not up to date enough to use the OnlyPost script you will be prompted to update it. Please do so if prompted.
  4. Close the terminal window when the script is done executing and you see the "[Process completed]" message at the bottom of your terminal screen.

Installing Requirements

  1. Open a new terminal window (click here if you forgot how to open a new terminal window -- read step 6).
  2. Copy and paste the red text below into the terminal and hit enter.

    3. cd Desktop/Scripts; zsh devToolsInstall.sh

  3. Enter your macOS user password when prompted.
  4. The terminal will display a list of what will be installed. Hit return (the enter button) when you are asked to do so.
  5. Close the terminal window when the requirements are finished installing and you see the "[Process completed]" message (this process might take a few minutes).

Setting Up Your creds.py File

  1. Open a new terminal window (click here if you forgot how to open a new terminal window -- read step 6).
  2. Copy and paste the red text below into the terminal and hit enter.

    3. cd Desktop/Scripts; zsh creds.sh

  3. Enter all of the information you are asked for. This information is case sensitive so please be mindful of that when entering things.
  4. Close the terminal window when the script is done executing and you see the "[Process completed]" message at the bottom of your terminal screen.
  5. If you entered any of your Reddit credentials incorrectly in the above steps, simply copy and paste the red text under step 2 into the terminal again and hit enter to update your credentials with the correct information.

A Final Note on Security

You have now successfully set up your OnlyPost script. Please do not share any of the files in the OnlyPost folder that you have saved onto your desktop with anyone. At this point those files contain your Reddit login crednentials and if you were to send them to someone for any reason or let them read the files for any reason they will be able to see your Reddit username, password, etc.
